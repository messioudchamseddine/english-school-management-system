# English School Management System

A Full-Stack web application for managing an English language school.

## Features

- Student Management
- Teacher Management
- Course Management
- Authentication & Authorization
- REST API
- Responsive Angular Dashboard

## Technologies

### Backend
- Java
- Spring Boot
- Spring Data JPA
- Spring Security
- MySQL

### Frontend
- Angular
- TypeScript
- HTML
- CSS
- Bootstrap

## Database
- MySQL

## Installation

### Backend

```bash
mvn spring-boot:run
```

### Frontend

```bash
npm install
ng serve
```
